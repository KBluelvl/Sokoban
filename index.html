Dans ce jeu vous devez mettre toutes les boites sur les cibles pour gagner.



Pour ce faire:

vous pouvez pousser les boites

Mais attention:

Vous ne pouvez pas pousser deux boites (ou plus) en même temps.


Vous ne pouvez pas tirer de boite.

Quand vous avez fini. Appuyer sur espace pour passer au niveau suivant.
